# OpenDebt dokumentation

Velkommen til dokumentationen for **OpenDebt** -- et open source-inddrivelsessystem til danske offentlige institutioner.

OpenDebt er designet til at erstatte ældre systemer (EFI/DMI) med en moderne, PSRM-kompatibel arkitektur baseret på microservices, Java 21, Spring Boot 3.3 og PostgreSQL 16.

---

## Dokumentationsoversigt

### [Teknisk dokumentation](technical/index.md)

Arkitekturbeskrivelse, udviklervejledning, API-reference og domænemodel. Skrevet på engelsk for det internationale udviklersamfund.

### [Fordringshaver](fordringshaver/index.md)

Vejledning til fordringshavere (kreditorer) der bruger Fordringshaverportalen eller M2M-integrationer til at indsende og administrere fordringer.

### [Skyldner](skyldner/index.md)

Vejledning til borgere (skyldnere) der bruger Skyldnerportalen til at se deres gældsoverblik, foretage betalinger og håndtere indsigelser.

### [Sagsbehandler](sagsbehandler/index.md)

Vejledning til sagsbehandlere i Gældsstyrelsen der arbejder med sagshåndtering, inddrivelsesskridt og fordringshåndtering.
