# Fordringshaverportalen

Velkommen til vejledningen for **Fordringshaverportalen** i OpenDebt.

Fordringshaverportalen er den digitale selvbetjeningsløsning, hvor fordringshavere (offentlige myndigheder og institutioner) kan indsende, administrere og følge deres fordringer til inddrivelse hos Gældsstyrelsen.

## Hvad kan du som fordringshaver?

- **Oprette og indsende fordringer** til inddrivelse
- **Følge fordringens livscyklus** fra registrering til indfrielse eller afskrivning
- **Se fordringens tidslinje** — en samlet kronologisk visning af alle hændelser på fordringen (betalingsposteringer, statusændringer, inddrivelsesskridt og meddelelser)
- **Regulere fordringer** (opskrivning, nedskrivning, tilbagekald)
- **Modtage underretninger** om dækninger, renter og statusændringer
- **Afstemme** dine fordringer mod afregninger og udligninger
- **Trække rapporter** over fordringer, dækninger og renter

## Adgang til portalen

Fordringshaverportalen er tilgængelig via:

- **Webportal**: Log ind med dit organisationslogin via Keycloak
- **M2M-integration**: Systemet kan forbindes direkte via REST API eller SOAP (se [M2M-integration](m2m-integration.md))
