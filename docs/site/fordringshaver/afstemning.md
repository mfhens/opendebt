# Afstemning og rapporter

Afstemning sikrer, at fordringshaveres regnskab stemmer overens med Gældsstyrelsens registreringer.

## Månedlig afstemning

Den månedlige afregningsmeddelelse er grundlaget for afstemning. Den indeholder:

- Alle dækninger (betalinger) i den forgangne måned
- CPR/CVR for de berørte skyldnere
- Beløb og datoer
- Fordringsreferencer

Sammenlign afregningsmeddelelsen med dine egne betalingsregistreringer for at verificere, at alle beløb stemmer.

## Rapporter i portalen

Fordringshaverportalen giver adgang til følgende rapporttyper:

| Rapport | Indhold |
|---------|---------|
| Fordringsoversigt | Alle aktive fordringer med status og saldo |
| Dækningsoversigt | Betalinger modtaget pr. fordring |
| Renteoversigt | Tilskrevne inddrivelsesrenter pr. fordring |
| Statustæller | Antal fordringer pr. livscyklustilstand |
| Månedlig afregning | Den formelle månedlige afregningsmeddelelse |

## Afstemningsproces

1. **Hent afregning**: Download den månedlige afregning fra portalen eller via API
2. **Sammenlign**: Match dækningsbeløb mod dit eget regnskab
3. **Identificer afvigelser**: Noter eventuelle forskelle
4. **Følg op**: Ved uoverensstemmelser, kontakt Fordringshaversupport

## Udligning som dagligt afstemningsværktøj

Hvis du modtager daglige udligningsmeddelelser, kan disse bruges til løbende afstemning af saldi. Udligningen viser saldobevægelser, så du altid har et opdateret billede af dine fordringers status.

!!! tip "Automatisk afstemning"
    Brug OCR-linjer ved oprettelse af fordringer for at muliggøre automatisk afstemning mellem betalinger og fordringer.
