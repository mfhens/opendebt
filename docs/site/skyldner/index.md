# Skyldnerportalen

Velkommen til vejledningen for **Skyldnerportalen** i OpenDebt.

Skyldnerportalen er den digitale selvbetjeningsløsning, hvor borgere (skyldnere) kan se deres gæld til det offentlige, foretage betalinger og håndtere indsigelser.

## Hvad kan du som skyldner?

- **Se dit gældsoverblik** med alle aktive fordringer
- **Logge ind sikkert** med MitID
- **Foretage betalinger** mod din gæld
- **Gøre indsigelse** mod en fordring du mener er forkert
- **Se renteregler** og forstå hvordan renter beregnes
- **Kontakte Gældsstyrelsen** ved spørgsmål
