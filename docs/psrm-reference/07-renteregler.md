# Renteregler

Inddrivelsesrente: **5,75%** (per 1. januar 2026). Simpel dag-til-dag rente af hovedstol. Inddrivelsesrenten dækkes altid forud for hovedstol (dækningsrækkefølge).

## Renteberegning kan IKKE ændres

Er den angivet forkert: tilbagekald med FEJL og indsend på ny.

## Standardopsætning

Se fordringstype-specifik opsætning i individuelle aftaler.

## Renter der afviger fra inddrivelsesrenten

Gælder fx: pantebrev, gældsbrev, udenretligt forlig, aftalt morarente, retsafgørelse med afvigende rente.

### Ingen rentetilskrivning
- Renteregel: 001 (RIM beregner)
- Rentesatskode: 98 (Ingen renteberegning)

### Anden rentetilskrivning
- Renteregel: 001
- Rentesatskoder:
  - 03: Ren merrentesats p.a. (angiv i feltet "Merrentesats")
  - 10: Morarente før 01.08.2002
  - 20: Procesrente
- Rentesats må ikke overstige den afvigende rente
- Kun dag-til-dag rente

### Egen rentetilskrivning
- Renteregel: 002 (Fordringshaver beregner)
- Rentesatskode: 99 (Fordringshaverrenter)
- Renter indsendes løbende som relaterede fordringer
- Kontakt fordringshaversupport for specifik fordringstype

## Undtagelser

- **Bøder** (straffelovsbøder): helt undtaget for inddrivelsesrente (GIL §5 stk 1)
- **Dagbøder** (skattekontrolloven, planloven): ER rentebærende
- **Gældsbrev/aftalt rente/retsafgørelse:** Kan aftale at Gældsstyrelsen beregner med angivet rentesats, eller fordringshaver beregner selv
- **Told:** EU-retsakt rentesats bruges

## Rentestop for uafklaret gæld (1. november 2024)

- Ingen inddrivelsesrenter på uafklarede gældsposter hvor skyldner ikke kan afdrage
- Allerede påløbne renter forbliver del af gælden
- Skyldner varsles hvis gæld gøres inddrivelsesparat igen
- Gælder IKKE for fordringer med anden rente end inddrivelsesrenten
- Gælder IKKE for fordringer med egen rentetilskrivning (fx lånefordringer)

## Fradragsret

Ingen fradragsret for inddrivelsesrenter siden 1. januar 2020.
