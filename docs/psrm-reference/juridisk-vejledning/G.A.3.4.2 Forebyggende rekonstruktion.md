# G.A.3.4.2 Forebyggende rekonstruktion

# Indhold

Dette afsnit beskriver reglerne om forebyggende rekonstruktion, deres formål og de forskellige faser i en forebyggende rekonstruktion. Derudover beskrives reglerne om tidlig varsling.
