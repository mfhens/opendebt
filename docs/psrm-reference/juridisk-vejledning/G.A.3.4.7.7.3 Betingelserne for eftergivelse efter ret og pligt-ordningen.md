# G.A.3.4.7.7.3 Betingelserne for eftergivelse efter ret og pligt-ordningen

# Indhold

Afsnittet handler om betingelserne for, at en skyldner kan få eftergivet sin gæld til det offentlige efter ret og pligt-ordningen.

Afsnittet indeholder:

  * Gældens størrelse ([G.A.3.4.7.7.3.1](https://info.skat.dk/data.aspx?oid=2387371&chk=220619 "Linket dirigerer dig til: G.A.3.4.7.7.3.1 Gældens størrelse"))
  * Ledighedskravet ([G.A.3.4.7.7.3.2](https://info.skat.dk/data.aspx?oid=2387372&chk=220619 "Linket dirigerer dig til: G.A.3.4.7.7.3.2 Ledighedskravet"))
  * Beskæftigelses-/uddannelseskravet ([G.A.3.4.7.7.3.3](https://info.skat.dk/data.aspx?oid=2387373&chk=220619 "Linket dirigerer dig til: G.A.3.4.7.7.3.3 Beskæftigelses-/uddannelseskravet")).



### Bemærk

Det tidligste tidspunkt betingelserne for eftergivelse efter ret og pligt-ordningen kan være opfyldt, er den 1. april 2025. Se mere herom i [G.A.3.4.7.7.3.3](https://info.skat.dk/data.aspx?oid=2387373&chk=220619 "Linket dirigerer dig til: G.A.3.4.7.7.3.3 Beskæftigelses-/uddannelseskravet") Beskæftigelses-/uddannelseskravet.
