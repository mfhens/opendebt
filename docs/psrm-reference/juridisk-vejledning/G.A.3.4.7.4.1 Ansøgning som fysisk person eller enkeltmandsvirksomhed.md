# G.A.3.4.7.4.1 Ansøgning som fysisk person eller enkeltmandsvirksomhed

**Indhold**

Afsnittet handler om fysiske personers og enkeltmandsvirksomheders ansøgning om eftergivelse.

Afsnittet indeholder:

  * Ansøgning om eftergivelse
  * ► ◄



**Ansøgning om eftergivelse**

►Gæld til det offentlige kan kun eftergives efter ansøgning fra skyldner. Der er ingen formkrav til en ansøgning om eftergivelse. Restanceinddrivelsesmyndigheden har dog udarbejdet blanketter, som skyldner kan anvende til brug for ansøgningen,. Disse kan findes på [www.gældst.dk](http://www.gældst.dk/)◄

► ◄

I forbindelse med behandlingen af ansøgningen er restanceinddrivelsesmyndigheden kun forpligtet til at lægge vægt på de oplysninger i ansøgningen, som vurderes at være reelle for skyldneren. Se [SKM2007.228.HR](https://info.skat.dk/data.aspx?oid=1622674&chk=220354 "Linket dirigerer dig til: Gældseftergivelse - skattegæld - frafaldelse af sagbehandfejl").
