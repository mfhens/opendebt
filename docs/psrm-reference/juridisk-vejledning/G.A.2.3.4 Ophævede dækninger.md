# G.A.2.3.4 Ophævede dækninger

# Indhold

Dette afsnit beskriver reglerne for håndtering af ophævede dækninger, som opstår ved nedskrivning eller annullation af en helt eller delvist dækket fordring, eller som følge af interne opskrivninger m.v.

Afsnittet indeholder:

  * Korrektionspuljen for PSRM ([G.A.2.3.4.1](https://info.skat.dk/data.aspx?oid=2387395&chk=220619 "Linket dirigerer dig til: G.A.2.3.4.1 Korrektionspuljen for PSRM"))
  * Korrektionspuljen for DMI ([G.A.2.3.4.2](https://info.skat.dk/data.aspx?oid=2387396&chk=220619 "Linket dirigerer dig til: G.A.2.3.4.2 Korrektionspuljen for DMI"))
  * Korrektionspuljen for tværgående lønindeholdelse ([G.A.2.3.4.3](https://info.skat.dk/data.aspx?oid=2436919&chk=220619 "Linket dirigerer dig til: G.A.2.3.4.3 Korrektionspuljen for tværgående lønindeholdelse"))
  * Interne opskrivninger ([G.A.2.3.4.4)](https://info.skat.dk/data.aspx?oid=2397930&chk=220619 "Linket dirigerer dig til: G.A.2.3.4.4 Interne opskrivninger")
  * Særligt om konverterede fordringer ([G.A.2.3.4.5](https://info.skat.dk/data.aspx?oid=2397932&chk=220619 "Linket dirigerer dig til: G.A.2.3.4.5 Særligt om konverterede fordringer")).
