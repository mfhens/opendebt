# G.A.3.4.7.4.2 Ansøgning som juridisk person

# Indhold

Afsnittet handler om juridiske personers ansøgning om eftergivelse.

Afsnittet indeholder:

  * Ansøgning om eftergivelse.



# Ansøgning om eftergivelse

► ◄

►Gæld til det offentlige kan kun eftergives efter ansøgning fra skyldner. Der er ingen formkrav til en ansøgning om eftergivelse. Restanceinddrivelsesmyndigheden har dog udarbejdet en blanket som juridiske personer kan anvende til brug for ansøgningen. Denne kan findes på [www.gældst.dk](http://www.gældst.dk/)◄

Eftergivelse af gæld til det offentlige kan kun meddeles juridiske personer, når sociale eller andre forhold i særlig grad taler derfor. Se [gældsinddrivelseslovens § 13, stk. 6](https://www.retsinformation.dk/eli/lta/2024/1063#P13). Kravet "når sociale eller andre forhold i særlig grad taler derfor" viser, at adgangen til eftergivelse for juridiske personer er betydeligt mere begrænset end for fysiske personer. Se mere om eftergivelse efter [gældsinddrivelseslovens § 13, stk. 6](https://www.retsinformation.dk/eli/lta/2024/1063#P13) i afsnit [G.A.3.4.7.3.3](https://info.skat.dk/data.aspx?oid=2323643&chk=220354 "Linket dirigerer dig til: G.A.3.4.7.3.3 Sociale eller andre forhold der i særlig grad taler for eftergivelse") Sociale eller andre forhold, der i særlig grad taler for eftergivelse

Juridiske personer, der kan søge om eftergivelse er:

  * Selskaber
  * Foreninger
  * Selvejende institutioner
  * Fonde
  * Eller lignende



Følgende kan ikke få eftergivet gæld:

  * Dødsboer
  * Konkursboer
  * Interessentskaber
  * Kommanditselskaber
  * Partrederier og lignende



Enkeltmandsvirksomheder skal søge om eftergivelse på samme måde som fysiskepersoner. Se [G.A.3.4.7.4.1](https://info.skat.dk/data.aspx?oid=2323647&chk=220619 "Linket dirigerer dig til: G.A.3.4.7.4.1 Ansøgning som fysisk person eller enkeltmandsvirksomhed") Ansøgning som fysisk person eller enkeltmandsvirksomhed
