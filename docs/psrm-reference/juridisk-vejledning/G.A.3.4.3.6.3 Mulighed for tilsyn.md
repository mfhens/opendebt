# G.A.3.4.3.6.3 Mulighed for tilsyn

# Indhold

Dette afsnit handler om muligheden for at underkaste skyldneren tilsyn, hvis denne bringer opfyldelsen af rekonstruktionen i fare.

Afsnittet indeholder:

  * Mulighed for tilsyn, hvis skyldners forhold bringer rekonstruktionen i fare.



# Mulighed for tilsyn, hvis skyldners forhold bringer rekonstruktionen i fare

Må rekonstruktionens opfyldelse antages at være bragt i fare ved skyldnerens forhold, kan skifteretten på begæring af fordringshaver bestemme, at skyldneren skal underkaste sig tilsyn af rekonstruktøren. Se [konkurslovens § 14 d](https://www.retsinformation.dk/eli/lta/2024/1162#P14d).

[Konkurslovens § 13 e, stk. 7](https://www.retsinformation.dk/eli/lta/2024/1162#P13e), der beskriver, hvad tilsynet skal påse, finder tilsvarende anvendelse. Se mere herom i [G.A.3.4.3.5.3](https://info.skat.dk/data.aspx?oid=2350791&chk=220619 "Linket dirigerer dig til: G.A.3.4.3.5.3 Skifterettens rolle") Skifterettens rolle

Tilsyn beskikkes ikke, før skyldneren er blevet hørt, og skifteretten kan ligeledes høre andre fordringshavere end den, der har begæret tilsynet.
