# G.A.3.4.7.6 Eftergivelsessagens afslutning

# Indhold

Dette afsnit handler om eftergivelsessagens afslutning. Herunder behandles afgørelsen, klageadgang og restanceinddrivelsesmyndighedens mulighed for at genoptage eller tilbagekalde afgørelsen om eftergivelse.
