# G.A.1.2 Snitflade mellem opkrævning og inddrivelse

# Indhold

Dette afsnit beskriver den lovbestemte snitflade mellem opkrævning og inddrivelse, der blev indført med [lov nr. 429 af 6. juni 2005 om opkrævning og inddrivelse af visse fordringer](https://www.retsinformation.dk/eli/lta/2005/429), og som er ført videre i [gældsinddrivelsesloven](https://www.retsinformation.dk/eli/lta/2024/1063).

Afsnittet indeholder:

  * Kompetenceforholdet mellem fordringshaver og restanceinddrivelsesmyndigheden ([G.A.1.2.1](https://info.skat.dk/data.aspx?oid=1949234&chk=220619 "Linket dirigerer dig til: G.A.1.2.1 Kompetenceforholdet mellem fordringshaver og restanceinddrivelsesmyndigheden"))
  * Opgaver og kompetencer i forbindelse med inddrivelsesarbejdet ([G.A.1.2.2](https://info.skat.dk/data.aspx?oid=1949236&chk=220619 "Linket dirigerer dig til: G.A.1.2.2 Opgaver og kompetencer i forbindelse med inddrivelsesarbejdet")).
