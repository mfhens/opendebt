# AKR-numre (Skyldner uden CPR/CVR)

AKR-nummer kan tildeles personer og virksomheder uden dansk CPR eller CVR.

## Påkrævede oplysninger

### Obligatorisk

- Navn (person eller virksomhed)
- Adresse
- Postnummer (hvis Danmark eller Grønland)
- By
- Landekode
- Landenavn

### Minimum ét identifikationsnummer

- Pasnummer
- Kørekortnummer
- TIN (Tax Identification Number)
- Social security nummer
- CRS-nummer
- Udenlandsk virksomhedsnummer
- EAN/GLN-nummer

## Proces

1. Download formular fra gaeldst.dk
2. Udfyld elektronisk
3. Send til fordringshaver@gaeldst.dk med 4-cifret fordringshaver-ID
4. Gældsstyrelsen opretter skyldner i PSRM
5. Formular returneres med tildelte AKR-numre
