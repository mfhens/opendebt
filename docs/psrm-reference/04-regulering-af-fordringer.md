# Regulering af fordringer

Fordringshavere kan til enhver tid op- og nedskrive samt tilbagekalde allerede indsendte fordringer.

**Vigtig:** Afstem altid saldoen på fordringen inden korrektioner for at undgå uoverensstemmelser (fx ved direkte indbetaling fra skyldner).
