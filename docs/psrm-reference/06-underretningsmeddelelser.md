# Underretningsmeddelelser

Gældsstyrelsen sender underretninger ved nye handlinger på fordringer i PSRM. Fungerer som kvitteringer. Kan IKKE hentes ældre end 3 måneder.

## Leveringsmetode

- **System-til-system:** Hent via service i eget debitorsystem
- **Portalbrugere:** Hent i Fordringshaverportalen
- **DMI (gammel portal):** Via e-Boks (digital post)

## Typer

### 1. Afregning
- Månedlig, sendes sidste hverdag
- Periode: sidste bankdag forrige måned → dagen før sidste bankdag indeværende
- Indhold: identifikationsnummer (CPR), beløb, dato, fordring
- Sendes til NemKonto

### 2. Udligning
- Daglig ved saldo-bevægelse (dækning eller saldostigning)
- Bruges til at holde saldi ajour (vigtigt for nedskrivning)

### 3. Allokering
- Daglig ved saldo-bevægelse
- Som udligning MEN viser fordeling mellem afdrag på hovedstol og dækning af renter
- Inkl. dækningsophævelser ved krydsende finansielle transaktioner
- Gensidig eksklusiv med udligning (vælg én)

### 4. Renter
- Månedlig rentetilskrivning
- Eller detaljeret daglig (kun system-til-system)
- Ved tilbagekald med rente-returnering: renter udspecificeres i kvittering

### 5. Afskrivning
- Fordring mistet retskraft: forældelse, konkurs, dødsbo, gældssanering
- Eller: åbenbart formålsløs / uforholdsmæssigt omkostningsfuld inddrivelse
- Indhold: fordringsID, beløb, evt. identifikationsnummer ved flere hæftere

### 6. Tilbagesend og Returnering
- Fordringshaver anmoder om tilbagesendelse
- Eller Gældsstyrelsen returnerer (fx manglende reaktion på klage)
- Underretter også om saldostigninger på tilbagekaldte fordringer (fra omkontering)
