# Teknisk Tilslutning og Adgang (Technical Connection and Access)

The system-to-system (S2S) solution enables automatic claim creation in PSRM, bypassing the manual Fordringshaverportalen workflow.

## Overview

- **Purpose:** Automate submission of claims directly from the creditor's debitor system to PSRM.
- **Suitable for:** High-volume creditors with many claims to submit.
- **Requires:** Technical setup where the creditor's debitor system communicates with PSRM via APIs.

## Setup Process

1. Typically requires involvement of the creditor's **system vendor**.
2. Gældsstyrelsen arranges **technical workshops** and **ad-hoc meetings** to coordinate integration.
3. Contact **fordringshaver@gaeldst.dk** to initiate the setup process.

## Existing S2S Users

Creditors already using the system-to-system solution should contact **fordringshaversupport** for:

- General questions about S2S functionality
- Technical issues or errors
- Testing new functionality or changes
