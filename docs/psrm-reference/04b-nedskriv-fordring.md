# Nedskriv fordring

Kan IKKE nedskrive mere end restsaldo i PSRM (afvises). Hold jer ajour via udligningsunderretninger.

## Årsagskoder

### INDB (Indbetalinger)
Indbetaling modtaget fra skyldner efter oversendelse. Gælder både ikke-posterede fra opkrævningsfasen og indbetalinger under inddrivelse.

- Kun hovedfordring nedskrives; inddrivelsesrente forbliver
- Virkningsdato = indbetalingsdato fra skyldner (obligatorisk)
- Samtidig indbetaling: PSRM bogføringstidspunkt afgør
- Hvis PSRM-indbetaling først: fordringshaver skal evt. refundere skyldner
- Hvis fordringshaver modtog først: MÅ IKKE udbetale direkte til skyldner - korriger forholdet, Gældsstyrelsen tilbagekalder afregning

### OANI (OpskrivningAnnulleretNedskrivningIndbetaling)
Annuller fejlagtig INDB-nedskrivning (fx ompostering flytter indbetaling til anden fordring).

### REGU (Reguleringer)
Nedskriv på baggrund af korrektion i eget system (fx berettiget reduktion).

- Virkningsdato = altid fordringens modtagelsesdato
- Brug begrundelsesfeltet
- For meget afregnet trækkes tilbage; inddrivelsesrenter på reguleret beløb nulstilles

### OONR (OpskrivningOmgjortNedskrivningRegulering)
Omgør fejlagtig REGU-nedskrivning.
